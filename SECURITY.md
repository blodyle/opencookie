# Security Policy

Please do not report security issues in public GitHub issues.

Security issues may include:

- Optional scripts rendering before consent.
- Consent choices being ignored or bypassed.
- Unsafe script loading behavior.
- XSS risks in examples or default UI.
- Storage bugs that corrupt or leak consent choices.
- Dependency vulnerabilities.

Use GitHub private vulnerability reporting if it is enabled for the repository. If it is not enabled yet, contact the maintainer privately before sharing details publicly.

OpenCookie is frontend-only. It does not provide legal advice, legal compliance guarantees, audit logs, hosted storage, or cross-device consent records.
